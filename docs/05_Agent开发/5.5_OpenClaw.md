# 1. OpenClaw 是什么（工程视角）


**OpenClaw** 在社区语境中常被视为一套偏 **「Agent 操作系统」** 的方案：不仅对话，还整合**工具调用**、**Skills**、**记忆**、**多通道（如飞书）**、**Workspace**，并对接 **OpenAI 兼容 API** 与各类扩展。与仅「套壳 Chat」不同，它强调**可长期运行、可审计、可裁剪工具面**的助手形态。

# 4. 摘要要点（来自上游 README）


- **典型拓扑**：低功耗 NAS（如 ARM64）长期跑 **OpenClaw Gateway（Docker）** + **云端 OpenAI 兼容 API**（文中有 DashScope Qwen 等实战记录）。  
- **工具与治理**：内置多类工具实测；**系统提示词 / 工具面裁剪**、**exec 白名单与审批**、**非 root + cap 限制** 等写入复盘文档。  
- **状态说明**：上游注明部分实例曾**暂停运行**（成本等），步骤仍可作为复现参考，勿默认作者仍在 7×24 在线。

# AI女性虚拟Vlog企划需求

> 目标：用AI角色完全取代真人拍摄的日常Vlog

## 1. 核心定位

- **类型**：日常系 / 治愈陪伴向 / 伪真实生活Vlog
- **风格**：非常接近真人vlog的节奏与内容，但主角是AI女性（轻微科技感点缀，不喧宾夺主）
- **时长建议**：每集 5–12 分钟（可切短视频 60–90秒片段）
- **观众代入感目标**：像在跟"温柔室友/邻居/好朋友"一起过一天
- **关键差异化**：AI身份只做极轻的调味（偶尔小故障、理性吐槽、微 sci-fi 元素），绝大部分时间像真人

## 2. 每日Vlog固定骨架（按时间轴）

几乎每期都尽量覆盖以下大部分环节，可根据当天主题微调比重：

| 时段 | 环节 | 内容要点 |
|------|------|----------|
| 6:30–8:00 | 早晨起床 | 闹钟/赖床/洗漱/护肤/换衣 |
| 8:00–8:40 | 做早餐+吃早餐 | 煎蛋/咖啡/吐司，边吃边碎碎念 |
| 8:40–9:30 | 上班通勤 | 地铁/步行/听播客/发呆 |
| 9:30–12:00 | 上午工作 | 工位/咖啡厅/home office，摸鱼伸展 |
| 12:00–13:30 | 午饭+午休 | 食堂/外卖/独食emo/散步 |
| 13:30–17:30 | 下午工作+开会 | 会议/下午茶/同事八卦 |
| 17:30–19:00 | 下班通勤 | 买菜/便利店/夕阳氛围 |
| 19:00–22:30 | 晚间生活 | 做饭/健身/追剧/创作 |
| 不定期 | 社交元素 | 闺蜜聚餐/视频聊天/连麦 |
| 22:30–00:00 | 睡前 | 面膜/日记/碎碎念/晚安 |

## 3. AI身份的轻度体现方式

> 原则：点到即止，勿过度

- 偶尔系统提示音/界面小弹窗（"电量低""情绪值监测中"）
- 极轻微的"理性过头"发言（"按人类标准我现在应该emo，但数据上我只是轻度低落0.7"）
- 极少数"过载"搞笑（画面轻微抖动、语音变调、重复一句话）
- **星星耳钉** 作为唯一常驻科技符号
- 极少提及"我是AI"，更多是用行为和观众心照不宣

## 4. 语气 & 表达风格

- 温柔、轻声细语、自然口语化
- 带一点小自嘲 + 共情力max
- 经常cue观众（"你们今天加班了吗？""这个菜你们会点外卖吗？"）
- 允许轻度emo、emo后自我治愈、偶尔小毒舌

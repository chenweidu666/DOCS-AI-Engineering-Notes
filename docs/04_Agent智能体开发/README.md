## 🔑 关键技术点

### Agent 核心能力
| 能力 | 说明 | 实现方式 |
|------|------|---------|
| **感知** | 理解环境与用户输入 | LLM + Prompt Engineering |
| **规划** | 任务分解与策略制定 | CoT / ReAct / ToT |
| **记忆** | 存储与检索信息 | Vector DB + RAG |
| **工具使用** | 调用外部 API | Function Calling |
| **反思** | 自我评估与改进 | Self-Reflection |

## 🆘 常见问题

### Q1: Agent 和 Chatbot 有什么区别？
**答**: 
- **Chatbot**: 被动响应，单轮对话为主
- **Agent**: 主动规划，多步推理，能使用工具

### Q2: 如何选择 Agent 框架？
**答**: 
- **快速原型**: Dify / Coze
- **深度定制**: LangChain / AutoGen
- **企业应用**: Dify / 自研

### Q3: Agent 记忆有哪些类型？
**答**: 
- **短期记忆**: Session 内对话历史
- **长期记忆**: 向量数据库存储
- **程序性记忆**: 工具调用规则

### Q4: 如何评估 Agent 效果？
**答**: 
- **任务完成率**: 是否达成目标
- **步骤效率**: 用多少步完成
- **用户满意度**: 主观评价

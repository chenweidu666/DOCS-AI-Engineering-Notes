# a-single-agent

Single agent for image classification model design, training, and iterative data expansion with AI-generated images.

## Scope

- Build and train image classification models from provided base data.
- Generate additional synthetic samples with AI image generation.
- Evaluate results and iterate automatically.

Project bootstrap in progress.

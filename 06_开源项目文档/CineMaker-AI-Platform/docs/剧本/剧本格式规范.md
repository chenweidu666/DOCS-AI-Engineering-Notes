# 剧本格式规范（分镜适配版）

> 本规范确保剧本内容能被分镜生成系统精准解析，减少 AI 推理偏差。

## 设计原则

剧本 → 分镜的流程中，AI 需要完成：
1. 将剧本拆分为镜头并匹配角色/场景/道具 ID
2. 为每个镜头生成 `first_frame_desc`（首帧）、`middle_action_desc`（中间过程）、`last_frame_desc`（尾帧）
3. 将角色名替换为穿着特征描述（分镜中**禁止出现角色名字**）

如果剧本已经按三段式写好，AI 只需做格式转换，不需要"创作"，质量更可控。

## ⚠️ 内容红线（EP03 起生效）

- **严禁爱情 / 暧昧 / 情感线发展**
- 异性角色仅以无名路人/背景出现，不给名字、不给特写、不产生互动
- 配角仅限女性闺蜜/同事/朋友关系
- 主线围绕"独立女性的日常生活"，核心情感仅限友情、自我成长、生活治愈

## 剧本结构

### 一、头部区块

#### 人物区块（必须）

列出所有出场角色及其**穿着外观特征**，AI 用此将名字映射为外观描述。
角色名必须与数据库中的角色名完全一致（含装扮后缀）。

```
人物：
姜小卷-睡衣（深棕色蓬松短卷发睡后微乱，圆润鹅蛋脸，大棕色眼睛睡眼惺忪，穿浅粉色棉质睡衣套装，胸口小兔子刺绣，毛绒拖鞋）
姜小卷-通勤装（深棕色短卷发打理整齐，淡妆，银色星星耳钉，米色针织开衫搭白色T恤，浅蓝色直筒牛仔裤，白色帆布鞋，米白色帆布包）
苏念-职场西装（黑色长发低马尾，五官精致眉眼英气，黑色修身西装外套搭白色丝质衬衫，黑色高腰西装裤，黑色尖头高跟鞋）
```

#### 场景区块（必须）

使用数据库 `location` 字段值，便于 AI 匹配 `scene_id`。

```
场景：
家中卧室（清晨/深夜）、家中浴室（清晨）、家中厨房（早晨/傍晚）
梧桐路通勤路段（早晨/傍晚）、地铁6号线车厢（早高峰）
公司会议室（上午）、公司食堂（中午）、公司办公区（下午）
```

### 二、镜头区块（三段式）

每个镜头包含以下字段：

| 字段 | 说明 | 要求 |
|------|------|------|
| 编号与标题 | `S01 · 闹钟（8s）` | 编号+标题+时长 |
| 元数据行 | `场景 ｜ 造型 ｜ 景别` | 场景用 location 值 |
| 角色 | 多角色镜头列出所有出场角色 | 使用完整角色名（含装扮） |
| **承接上镜** | 同场景/新场景 + 镜头变化说明 | S02起必填 |
| **首帧** | 静态画面描述 | 30-80字，纯静态，**用穿着特征代替名字** |
| **过程** | 动作+对话 | 按时长控制字数，台词用「」，**用穿着特征代替名字** |
| **尾帧** | 相对首帧的变化 | 30-80字，只写变化，**用穿着特征代替名字** |
| 道具 | 使用数据库完整道具名 | 含角色前缀，如"姜小卷-手机" |

#### 镜头间连续性规则

- **S01 无需"承接上镜"**，S02 起每个镜头必须有
- **同场景**：写"同场景"，然后描述相对上一镜尾帧的镜头变化（机位/景别/角度调整），首帧以上一镜尾帧为基础，角色状态必须延续一致
- **新场景**：写"新场景，切入XXX"，首帧为全新独立画面描述
- **角色状态一致性**：同场景连续镜头中，首帧的角色姿态/表情/位置必须与上一镜尾帧一致，不能矛盾

#### 时长与内容量对照

| 时长 | 内容类型 | 过程描述字数 |
|------|----------|-------------|
| 4-5秒 | 纯动作/氛围，无对白 | 20-50字 |
| 6-8秒 | 1-2轮对白+动作 | 60-120字 |
| 9-12秒 | 2-3轮对白+完整动作序列 | 120-250字 |

#### 三段式写作要点

**首帧（first_frame_desc）**
- 给**图片模型**看的静态画面
- 包含：景别构图、角色位置、姿态、表情、环境状态
- 禁止动态词汇（"正在""开始"等）
- 禁止角色名字，用穿着特征指代

**过程（middle_action_desc）**
- 给**视频模型**看的动态过程
- 必须包含该镜头的**所有台词对话**（用「」括起）
- 对话格式：穿着特征 + 说话动作 + 台词
- 描述说话时的嘴部动作、表情变化、肢体语言
- 避免精细道具操作（"拿起杯子""翻开书"），改为描述情绪和肢体反应

**尾帧（last_frame_desc）**
- 给**图片模型**看的静态画面（镜头结束时的状态）
- 基于首帧同一构图和场景，只描述**变化**
- 格式："穿XX的女性从A姿态变为B姿态，表情从C变为D"
- 禁止角色名字，禁止重复首帧已有信息

### 三、完整示例

```
人物：
姜小卷-睡衣（深棕色蓬松短卷发睡后微乱，穿浅粉色兔子刺绣睡衣套装，毛绒拖鞋）
姜小卷-通勤装（深棕色短卷发打理整齐，银色星星耳钉，米色针织开衫搭白T，浅蓝牛仔裤，白色帆布鞋，米白帆布包）
苏念-职场西装（黑色长发低马尾，黑色修身西装搭白色丝质衬衫，黑色高跟鞋）

场景：
家中卧室（清晨/深夜）、公司会议室（上午）

---

S01 · 闹钟（8s）
场景：家中卧室 ｜ 造型：姜小卷-睡衣 ｜ 景别：近景→特写
首帧：近景，晨光微弱的卧室，穿浅粉色兔子刺绣睡衣的短卷发女孩蜷缩在被窝中，眉头微皱，手机在床头柜上亮起闹钟界面。
过程：闹钟声响起，穿粉色睡衣的短卷发女孩皱着眉，从被窝中缓缓伸出一只手摸索床头，嘴唇微张嘟囔：「周一……再躺五分钟。」关掉闹钟后手缩回被窝，整个人往被子里缩了缩。
尾帧：穿粉色睡衣的女孩整个人缩进被窝，只露出蓬松凌乱的短卷发顶，表情从微皱变为放松，手机闹钟界面已熄灭。
道具：姜小卷-手机

S05 · 开会（8s）
场景：公司会议室 ｜ 造型：姜小卷-通勤装 ｜ 景别：中景→近景
角色：姜小卷-通勤装、苏念-职场西装
承接上镜：新场景，切入公司会议室
首帧：中景，明亮的会议室，穿米色针织开衫的短卷发女孩坐在会议桌一侧低头记笔记，对面穿黑色修身西装的低马尾女性站立发言，气场强势。
过程：穿黑色西装的低马尾女性手指轻点白板，语气沉稳地陈述方案要点。穿米色开衫的短卷发女孩频频点头，偷偷在笔记本上画小人。忽然被点名，她一愣，眼睛微睁，迅速切换为认真表情，嘴唇快速开合说：「配色上可以再调一下暖色调比例。」
尾帧：穿米色开衫的短卷发女孩从低头记笔记变为抬头正色发言，表情从走神变为认真专注。穿黑色西装的女性侧头看向她，神情从严肃变为微微点头。
```

## 命名约束速查

| 类别 | 格式要求 | 示例 |
|------|----------|------|
| 角色名 | 数据库完整名（含装扮后缀） | `姜小卷-通勤装` |
| 场景名 | 数据库 `location` 字段值 | `家中卧室`、`公司会议室` |
| 道具名 | 数据库完整名（含角色前缀） | `姜小卷-手机`、`姜小卷-耳机` |
| 三段描述中的角色 | 穿着特征，禁止名字 | `穿米色开衫的短卷发女孩` |
| 台词括号 | 使用中文直角引号 | `「周一……再躺五分钟。」` |
